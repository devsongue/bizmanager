## Introduction

DevSongue Suite est une application de gestion multi-entreprise moderne conçue pour simplifier la gestion de vos activités commerciales. Cette suite complète vous permet de gérer efficacement vos ventes, dépenses, produits, clients, fournisseurs et employés depuis une interface unifiée.

### Fonctionnalités principales
- Gestion multi-entreprise : Gérez plusieurs entreprises à partir d'une seule interface
- Modules complets : Clients, Employés, Produits, Fournisseurs, Ventes, Dépenses, Rapports et Paramètres
- Panneau administratif : Pour les tâches administratives et la vue d'ensemble
- Contrôle d'accès basé sur les rôles : Rôles Administrateur et Gestionnaire avec autorisations appropriées
- Tableau de bord en temps réel : Aperçu financier avec graphiques et statistiques

## Connexion à l'application

1. Ouvrez votre navigateur web et accédez à l'adresse de l'application
2. Sur la page de connexion, saisissez vos identifiants :
   - **Email** : admin@devsonguesuite.ci
   - **Mot de passe** : password123
3. Cliquez sur le bouton "Se connecter"
4. Vous serez automatiquement redirigé vers le tableau de bord

**Remarque** : Pour des raisons de sécurité, il est fortement recommandé de changer le mot de passe par défaut lors de votre première connexion.

## Tableau de bord

Le tableau de bord est votre point de départ principal dans l'application. Il fournit un aperçu en temps réel des performances de votre entreprise.

### Éléments du tableau de bord
- **Statistiques clés** : Revenu total, Dépenses totales, Bénéfice total, Clients actifs
- **Graphique mensuel** : Visualisation des ventes et dépenses par mois
- **Ventes récentes** : Liste des dernières transactions
- **Alertes de stock faible** : Notifications sur les produits dont le stock est faible

### Filtrage par date
Vous pouvez filtrer les données affichées en utilisant le filtre de dates situé en haut à droite du tableau de bord.

## Gestion des ventes

La section Ventes vous permet d'enregistrer et de suivre toutes les transactions commerciales.

### Enregistrer une nouvelle vente
1. Cliquez sur "Ventes" dans le menu de navigation
2. Cliquez sur le bouton "Nouvelle vente"
3. Remplissez le formulaire avec les informations suivantes :
   - Client (sélectionnez dans la liste ou ajoutez un nouveau)
   - Produit (sélectionnez dans la liste)
   - Quantité vendue
   - Prix unitaire
   - Remise (si applicable)
   - Type de vente (Détail, Gros, Crédit, Espèces)
   - Méthode de paiement
4. Cliquez sur "Enregistrer la vente"

### Consulter l'historique des ventes
- Toutes les ventes sont affichées dans un tableau avec pagination
- Vous pouvez trier les ventes par date, client, produit ou montant
- Utilisez les filtres pour rechercher des ventes spécifiques

## Gestion des dépenses

La section Dépenses vous permet de suivre toutes les sorties financières de votre entreprise.

### Enregistrer une nouvelle dépense
1. Cliquez sur "Dépenses" dans le menu de navigation
2. Cliquez sur le bouton "Nouvelle dépense"
3. Remplissez le formulaire avec les informations suivantes :
   - Date de la dépense
   - Catégorie (ex: Salaire, Location, Matériel, etc.)
   - Description détaillée
   - Montant
   - Méthode de paiement
4. Cliquez sur "Enregistrer la dépense"

### Consulter l'historique des dépenses
- Toutes les dépenses sont affichées dans un tableau avec pagination
- Vous pouvez trier les dépenses par date, catégorie ou montant
- Utilisez les filtres pour rechercher des dépenses spécifiques

## Gestion des produits

La section Produits vous permet de gérer votre inventaire.

### Ajouter un nouveau produit
1. Cliquez sur "Produits" dans le menu de navigation
2. Cliquez sur le bouton "Nouveau produit"
3. Remplissez le formulaire avec les informations suivantes :
   - Nom du produit
   - Catégorie
   - Code SKU (facultatif)
   - Code-barres (facultatif)
   - Stock actuel
   - Stock minimum
   - Prix d'achat
   - Prix de détail
   - Prix de gros
   - Fournisseur (sélectionnez dans la liste)
4. Cliquez sur "Enregistrer le produit"

### Mettre à jour un produit
- Cliquez sur l'icône d'édition à côté du produit dans la liste
- Modifiez les informations nécessaires
- Cliquez sur "Mettre à jour le produit"

## Gestion des clients

La section Clients vous permet de gérer votre base de clients.

### Ajouter un nouveau client
1. Cliquez sur "Clients" dans le menu de navigation
2. Cliquez sur le bouton "Nouveau client"
3. Remplissez le formulaire avec les informations suivantes :
   - Nom complet
   - Coordonnées
   - Téléphone
   - Email
   - Adresse
   - Entreprise (facultatif)
   - Notes (facultatif)
4. Cliquez sur "Enregistrer le client"

### Consulter la liste des clients
- Tous les clients sont affichés dans un tableau avec pagination
- Vous pouvez trier les clients par nom, date de dernière commande ou points de fidélité
- Utilisez la recherche pour trouver un client spécifique

## Gestion des fournisseurs

La section Fournisseurs vous permet de gérer vos relations avec les fournisseurs.

### Ajouter un nouveau fournisseur
1. Cliquez sur "Fournisseurs" dans le menu de navigation
2. Cliquez sur le bouton "Nouveau fournisseur"
3. Remplissez le formulaire avec les informations suivantes :
   - Nom du fournisseur
   - Produits fournis
   - Coordonnées
   - Email
   - Téléphone
   - Adresse
   - Description
   - Types de produits
   - Notes (facultatif)
4. Cliquez sur "Enregistrer le fournisseur"

### Consulter la liste des fournisseurs
- Tous les fournisseurs sont affichés dans un tableau avec pagination
- Utilisez la recherche pour trouver un fournisseur spécifique

## Rapports et analyses

La section Rapports fournit des analyses détaillées sur les performances de votre entreprise.

### Types de rapports disponibles
- **Rapport de ventes** : Analyse des revenus par période, produit ou client
- **Rapport de dépenses** : Suivi des dépenses par catégorie ou période
- **Rapport de stock** : État des stocks et alertes de réapprovisionnement
- **Rapport de bénéfices** : Analyse de la rentabilité par produit ou période

### Générer un rapport
1. Cliquez sur "Rapports" dans le menu de navigation
2. Sélectionnez le type de rapport souhaité
3. Configurez les paramètres (période, filtres, etc.)
4. Cliquez sur "Générer le rapport"
5. Vous pouvez exporter le rapport au format PDF ou Excel

## Gestion des employés

Cette section est accessible uniquement aux utilisateurs avec le rôle Administrateur.

### Ajouter un nouvel employé
1. Cliquez sur "Employés" dans le menu de navigation
2. Cliquez sur le bouton "Nouvel employé"
3. Remplissez le formulaire avec les informations suivantes :
   - Nom complet
   - Email
   - Rôle (Administrateur ou Gestionnaire)
   - Téléphone
   - Adresse
4. Cliquez sur "Enregistrer l'employé"

### Gérer les permissions
- Les administrateurs ont accès à toutes les fonctionnalités
- Les gestionnaires ont accès aux modules de base mais pas au panneau administratif

## Paramètres et entreprises

Cette section est accessible uniquement aux utilisateurs avec le rôle Administrateur.

### Gérer les entreprises
1. Cliquez sur "Entreprises" dans le menu de navigation
2. Vous pouvez créer, modifier ou supprimer des entreprises
3. Chaque entreprise peut avoir ses propres paramètres, produits, clients, etc.

### Paramètres de l'entreprise
- Informations de base (nom, adresse, coordonnées)
- Paramètres de notification
- Paramètres d'intégration
- Préférences de devise et de langue

## Panneau d'administration

Cette section est accessible uniquement aux utilisateurs avec le rôle Administrateur.

Le panneau d'administration fournit une vue d'ensemble de toutes les entreprises gérées par le système.

### Fonctionnalités du panneau admin
- Vue d'ensemble de toutes les entreprises
- Statistiques globales
- Gestion des utilisateurs
- Historique d'audit
- Configuration système

## Déconnexion

Pour vous déconnecter de l'application :
1. Cliquez sur votre avatar ou nom d'utilisateur dans le coin supérieur droit
2. Sélectionnez "Déconnexion" dans le menu déroulant
3. Vous serez redirigé vers la page de connexion
